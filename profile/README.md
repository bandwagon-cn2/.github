**CN2 GT（普通版）**：去程走普通163网络，回程才走CN2。说白了，只有一半时间在走优化线路，高峰期效果打折。

**CN2 GIA（高级版）**：去程和回程都走CN2，而且是Premium级别，不和普通用户抢资源。这就是很多人说的"电信精品线路"，延迟低、稳定性强，晚上高峰期体验明显优于GT。
